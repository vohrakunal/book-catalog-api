# Books Catalog API

## To start API
- create the environment file by copying the env.example to ".env"
- replace the values of .env file according to the environment
- npm install
- start application
    - npm run dev (Development mode)
    - npm start (Prod mode)
    - npm build (to create a production build)


## API Routes

### Get Books
